## Reporting a Vulnerability

We take the security of this project seriously. If you find any vulnerability, please report it immediately.

### How to Report

Please do **NOT** open a public issue. Instead, please send an email to [YOUR_EMAIL@example.com] or use GitHub's private vulnerability reporting feature if available.

We will try to acknowledge your report within 48 hours and provide a timeline for the fix.

### Disclosure Policy

We follow a responsible disclosure policy:
1. You report the vulnerability privately.
2. We investigate and fix the vulnerability.
3. We release a security update.
4. We publicly announce the vulnerability and credit you (if you wish).

Thank you for helping keep this project safe!
